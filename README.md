# 🎉 Классификация овощей

## 📝 Описание

Данный репозиторий содержит проект для классификации рукописных или загружаемых изображений. В качестве веб-фреймворков применяются **Streamlit** (интерактивный интерфейс для пользователя) и **FastAPI** (серверная часть для обработки изображений). Пользователи могут картинку прямо в браузере или загрузить изображение со своего устройства. Приложение обрабатывает изображение и предсказывает, какой овощ изображён на картинке, используя предварительно обученную модель машинного обучения.

**Пользователи могут:**

- ✍️ Нарисовать овощ прямо в браузере.
- 📁 Загрузить изображение овоща со своего устройства.

**Ссылка на развернутое приложение: https://vegetables-predict.streamlit.app**

## 📁 Структура проекта

- **`streamlit_app.py`**  
  Приложение на Streamlit, позволяющее:
  - 📤 Загрузить изображение или ✏️ нарисовать его в интерактивном холсте.
  - 🔄 Отправить изображение на сервер с помощью HTTP-запроса и вывести результат классификации.

- **`app.py`**  
  Серверная часть на базе FastAPI:
  - 📥 Принимает изображение в формате `UploadFile`.
  - 🎯 Использует предобученную модель для вывода предсказания.

- **`best_model.keras`**  
  Сериализованная модель машинного обучения (keras-файл).

- **`requirements.txt`**  
  📜 Список зависимостей, необходимых для работы проекта.
